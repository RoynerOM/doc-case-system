Formulario Asignación de Casos

En este formulario asignamos un caso y tiene los siguientes campos:


Consecutivo

Al guardar un caso, a este se le asignará un consecutivo, al inicio el campo no mostrará nada, después de guardar un caso en el consecutivo se mostrará un caso por delante del que esta, es decir, tenemos un caso que sería R-1, en el consecutivo se mostrará como R-2...etc.

NOTA: Este campo no se puede modificar y es auto incrementable



Fecha

En este campo podemos seleccionar la fecha en que fue asignado el caso



Código

El código le pertenece a una junta, en este campo escribimos un código válido, presionando la tecla “Enter”, si el código es válido se llenarán los campos de “Jta/Educ/Adm” (el nombre de una junta) y Circuito (circuito de una junta) que estén asociados a ese código



Medio Receptivo

En este campo se desplegará una lista con las formas de recepción que podrá elegir, basta con dar un click en un dato de la lista



Asignado a

Aquí se le desplegará una lista de personas a las que se le asignará el caso


Usamos el botón guardar cuando se hayan llenados todos los campos y este como su nombre indica guardar ese caso. El campo "buscar caso" buscará el consecutivo que se le digite y podrá reasignar un caso a otra persona, luego proceda a tocar el botón reasignar caso para completar la acción




Detalles de casos

En este formulario se detalla información de los todos los casos asignados como por ejemplo: A que institución pertenece,a quién se le asignó el caso, la fecha en que se asignó etc...



Funciones del formulario de detalles de casos:

  • Filtros de búsquedas

    Estas funciones permiten al usuario realizar busquedas de casos como por ejemplo: busqueda por consecutivo,por persona, por código de institución y por recepción.



    • Por Consecutivo

      Muestra el caso que coincida con el consecutivo.

    • Por persona

      Muestra la lista de casos que fueron asignados a la persona seleccionada.

    • Por circuito

      Muestra la lista de todos los casos que pertenecen a un circuito.

    • Por recepción

      Muestra la lista de todos los casos que pertenecen a un circuito.


Agregar y ver observación del caso seleccionado

Para ver o agregar la observación de un caso hay que darle click al siguiente botón , luego aparecera un formulario donde se podrá agregar el número de acta, fólio, fecha, información de documentación faltante, observación del caso etc...

Formulario:


Podrá agregar la observación dando click al botón de Guardar observación este mismo botón servirá para modifcar, en caso de que se necesite editar la observación

Ya por guardada la observación puede indicar que el caso está Por entregar, dando click en botón de Marcar entrega pendiente

En caso de que quiera pasar la información de la observación agregada puede exportarla a un archivo word,excel o pdf dando click en el botón Generar Boleta


Refrecar la información de la tabla

Después de revisado el caso deberá refrescar o actualizar la tabla para que cargue los ultimos cambios. Botón
